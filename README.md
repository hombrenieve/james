# james
Custom Dockerfile images for miscelaneouos purposes

## Structure

### latexmaker
The purpose of this image is the generation of latex documentation.

### meca
gtypist with spanish language support to learn typing

### desktop-containers
Disposable desktop environments running in Podman containers, accessible via VNC. Useful for testing tiling window managers and other Linux desktops without affecting your current installation. See [desktop-containers/README.md](desktop-containers/README.md) for usage and setup details.

Available tests:
- **sway** — Headless sway tiling WM on Arch Linux
